## O que é
Plataforma CRM + agentes de IA. Produto principal da Evolution API LTDA.

## Serviços
- **evo-ai-crm** — CRM principal (Rails + Vue.js)
- **evo-ai-frontend** — Frontend enterprise
- **evo-ai-frontend-community** — Frontend open source
- **evo-auth-service** — Serviço de autenticação (Ruby on Rails)
- **Evolution Hub** — Hub de integrações
- **Bot Runtime Service** — Orquestração de chatbot (Go standalone)
