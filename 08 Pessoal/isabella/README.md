# Isabella — Health Check-ins

## Estrutura padrão
- `baseline.md` → baseline inicial consolidado
- `evolution.md` → evolução consolidada (baseline → atual)
- `exams.md` → exames e observações médicas
- `treatment.md` → tratamento/rotina atual
- `reports/YYYY-MM-DD.md` → check-ins semanais individuais
- `data/raw/YYYY-MM-DD.csv` → dados brutos de balança
- `dashboard.html` e `dashboard.pdf` → artefatos ativos (Drive replace)
- `dashboards/dashboard-isabella-YYYY-MM-DD.pdf` → histórico de PDFs

## Compatibilidade legada
- `report.md` mantido como legado de consolidação inicial.
- `csv-raw/` mantido temporariamente para compatibilidade.
