# Davidson — Health Check-ins

## Estrutura padrão
- `baseline.md` → baseline inicial consolidado
- `evolution.md` → evolução consolidada (baseline → atual)
- `exams.md` → exames e observações médicas
- `treatment.md` → tratamento/rotina atual
- `reports/YYYY-MM-DD.md` → check-ins semanais individuais
- `data/raw/YYYY-MM-DD.csv` → dados brutos de balança
- `dashboard.html` e `dashboard.pdf` → artefatos ativos (opcional por pessoa)
- `dashboards/*.pdf` → histórico de PDFs

## Observação
- O histórico principal atual do Davidson está em `../../reports/` e em `evolution.md`.
- A pasta `reports/` individual foi preparada para padronização daqui em diante.
