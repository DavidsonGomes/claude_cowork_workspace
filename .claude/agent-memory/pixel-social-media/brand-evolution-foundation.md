## Posicionamento estrategico atual (Visao.md v2.2, marco 2026)

A Evolution e um ecossistema de IA, automacao e educacao — nao apenas uma ferramenta tecnica. Os tres pilares sao: Evo AI (plataforma), Evo Academy (educacao/certificacao) e Programa de Parceiros (white-label).

**Tagline aprovada para comunicacao:** "A tecnologia que muda negocios, agora ao alcance de todos."
(Esta no Visao.md — usar SEMPRE. Nao usar "Foundation, all in one" que esta obsoleta no brand guide.)

**Value prop correto:** open source + educacao + parceiros + resultado pratico
(Remover "baixo custo" — posiciona erroneamente como marca de preco barato.)

## Voz da marca

- Tom: profissional, tecnico, acessivel, orientado a solucao
- Deve comunicar: transformacao de carreira/negocio, nao apenas funcionalidades
- Deve soar como: parceiro estrategico, nao fornecedor de software
- Ancorar em valores: Liberdade, Transparencia ("Quem sabe joga limpo"), Brio, Sonho
