## Uavoip — contexto (reunião 31/03/2026)

**Pessoas:**
- **Flavio:** Sócio da Uavoip, líder comercial
- **Léo:** Contato técnico/financeiro na Uavoip

### Reestruturação financeira da Uavoip
- A Uavoip passou por uma reestruturação comercial/financeira para sobreviver ao crescimento (~15k conexões ativas com precificação insustentável).
- Todos os acordos com clientes e parceiros foram renegociados.

### Renegociação de comissão
- **Decisão:** Comissão de Davidson para o cliente **Icon** reduzida para **7%**.
- **Motivo:** Conta Icon não era lucrativa para a Uavoip com os termos antigos.
- Davidson aceitou; aguarda extrato financeiro atualizado da conta Icon vindo da Léo.

### Novo modelo de revenda
- Davidson se torna **revendedor**: usa nova tabela de preços escalável, adiciona sua margem e fatura clientes diretamente.
- Retém controle total sobre relacionamento e precificação dos clientes.

### Integração Webphone (oportunidade técnica)
- Uavoip ofereceu biblioteca React plug-and-play de **webphone** para integrar ao Evo CRM.
- Suporta: API oficial e não-oficial do WhatsApp + SIP para chamadas tradicionais.
- Implementação: via CDN ou import direto React; UI customizável via `window.voip` API.
- **Potencial:** Substituir sistemas PABX — driver significativo de crescimento.
- Léo vai enviar documentação (GitHub) para Davidson avaliar e colocar no roadmap futuro.
