## Rappet

**O que é:** Plataforma de criação de landing pages / páginas de captura.
**Quem usa:** Diego Lerina, para criar a página de captura do Evolution Summit 01.
**Contexto:** A página de captura do Summit foi construída no Rappet. O backend (integração de formulário, grupo WhatsApp) foi configurado por Davidson.

Não é uma ferramenta do ecossistema Evolution — é uma plataforma de marketing externa.
