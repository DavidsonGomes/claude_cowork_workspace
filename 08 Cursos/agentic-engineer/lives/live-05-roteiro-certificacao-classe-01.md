# Live 05 — Conclusão da Classe 1: do caos ao sistema

## Objetivo
Consolidar todo o aprendizado da Classe 1 em um sistema operacional simples, replicável e pronto para transição à Classe 2.

## Estrutura (75-90 min)

1) Recap ultra curto das 4 lives (10 min)
- Live 1: Visão
- Live 2: Prática
- Live 3: Customização
- Live 4: Build real

2) Framework final da Classe 1 (20 min)
- Mapa final em uma tela:
  Problema -> Spec mínima -> Build -> Review -> Deploy -> Iteração
- O que é obrigatório em qualquer projeto
- O que pode variar sem perder qualidade

3) Consolidação com caso único (25-30 min)
- Pegar o projeto da Live 4
- Organizar versão final
- Documentar decisões e trade-offs

4) Kit final do aluno (10-15 min)
- Checklist oficial
- Definition of Done
- Rubrica simplificada para autoavaliação
- Plano de evolução para Classe 2

5) Fechamento e transição (10 min)
- Erros que mais travam na Classe 1
- Sinal objetivo de prontidão para Classe 2
- Próximo passo claro

## Critério de conclusão da Classe 1
"Consegue repetir esse fluxo 3x com consistência e entrega funcional."

## CTA
"Se você já entrega com esse padrão, está pronto para orquestração da Classe 2."
