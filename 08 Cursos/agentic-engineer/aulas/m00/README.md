# M00 — Fundamentos de Agentic Engineering

## Objetivo do módulo
Ao final deste módulo, o aluno vai entender o que é um agente, qual é o papel do humano como orquestrador e como operar pelo fluxo mínimo (problema → spec → build → review → deploy) em vez de improvisar.

| # | Aula | Duração | Formato | Status |
|---|------|---------|---------|--------|
| 1 | O que é um Agente | 15 min | vídeo | draft |
| 2 | O que é a Camada Agentic (Application Layer) | 15 min | vídeo | draft |
| 3 | O papel do humano como orquestrador | 15 min | vídeo | draft |
| 4 | O fluxo mínimo: problema → spec → build → review → deploy | 15 min | vídeo + prática | draft |

## Entregável do módulo
Microfluxo documentado: o aluno descreve em 1 página o fluxo que vai usar no projeto do curso.
