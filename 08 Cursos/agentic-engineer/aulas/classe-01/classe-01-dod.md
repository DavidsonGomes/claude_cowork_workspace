# Classe 1 — Definition of Done (DoD)

A Classe 1 é considerada concluída quando TODOS os itens abaixo forem verdadeiros:

1. Existe um microprojeto funcional entregue.
2. O projeto foi conduzido por método (briefing -> PRD -> épicos -> stories -> execução -> review).
3. Há rastreabilidade mínima (commits claros + evidência de revisão).
4. O aluno demonstrou capacidade de corrigir problema sem abandonar o fluxo.
5. Há documentação final com:
   - objetivo
   - decisões principais
   - trade-offs
   - próximos passos

## Não é Done se:
- houver apenas demo sem artefatos de processo;
- houver código/resultado sem review;
- não for possível explicar como reproduzir a entrega.
