# Agentic Engineer

## Visão Geral
Curso prático para transformar devs plenos/sêniores e founders solos em Comandantes de Computação. O aluno aprende a operar o fluxo completo spec → build → review → deploy → governança usando agentes de IA com disciplina de engenharia real.

## Público-alvo
Dev pleno/sênior ou founder solo que já usa ferramentas de IA mas quer sair do improviso e entregar produto no ar com rastreabilidade e qualidade.

## Pré-requisitos
- Experiência básica com desenvolvimento de software
- Familiaridade com Git e linha de comando
- Conta no GitHub

## Objetivos de aprendizagem
Ao final do curso, o aluno será capaz de:
1. Transformar um briefing em PRD, épicos e stories com critério de aceite
2. Executar implementação incremental com revisão formal a cada entrega
3. Operar testes, lint e documentação como parte do fluxo — não como extras
4. Publicar e validar uma entrega em ambiente real
5. Aplicar governança mínima com rastreabilidade auditável
