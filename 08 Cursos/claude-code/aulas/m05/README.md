# M05 — Operação no Dia a Dia

## Objetivo do módulo
Ao final deste módulo, o aluno vai saber usar Claude Code eficientemente para debug orientado a hipótese, refatoração segura, documentação útil e gestão de backlog técnico.

## Aulas

| # | Aula | Duração | Formato | Status |
|---|------|---------|---------|--------|
| 1 | Debug assistido por hipótese | 20 min | vídeo + prática | draft |
| 2 | Refatoração com segurança | 20 min | vídeo + prática | draft |
| 3 | Documentação útil (sem burocracia) | 15 min | vídeo | draft |
| 4 | Gestão de backlog técnico | 15 min | vídeo + prática | draft |

## Entregável do módulo
1 bug resolvido com hipótese documentada + 1 refatoração com testes antes/depois + backlog técnico organizado.

## Critérios de aprovação
- [ ] Debug com hipótese explícita antes de qualquer mudança de código
- [ ] Refatoração executada sem regressão (testes passando antes e depois)
- [ ] Backlog técnico com pelo menos 3 itens priorizados e justificados
