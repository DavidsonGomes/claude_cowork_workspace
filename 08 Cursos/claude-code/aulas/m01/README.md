# M01 — Fundamentos e Setup

## Objetivo do módulo
Ao final deste módulo, o aluno vai entender o que é Claude Code na prática (além do básico), ter o ambiente mínimo configurado e executar o primeiro fluxo básico de trabalho sem erros comuns.

## Aulas

| # | Aula | Duração | Formato | Status |
|---|------|---------|---------|--------|
| 1 | O que é Claude Code na prática | 15 min | vídeo | draft |
| 2 | Instalação e ambiente mínimo | 20 min | vídeo + prática | draft |
| 3 | Fluxo básico de trabalho | 15 min | vídeo + prática | draft |
| 4 | Erros comuns no início — e como evitar | 15 min | vídeo | draft |

## Entregável do módulo
Ambiente configurado e funcionando + primeiro fluxo básico executado com resultado documentado.

## Critérios de aprovação
- [ ] Claude Code instalado e operacional no ambiente do aluno
- [ ] Primeiro fluxo básico executado com evidência (screenshot ou log)
- [ ] Erros comuns do módulo identificados e documentados
