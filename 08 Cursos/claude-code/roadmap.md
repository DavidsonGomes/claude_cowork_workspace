# Claude Code Curso — Roadmap

## Estado atual
- Curso criado neste tópico para trilha dedicada de Claude Code.
- Estrutura inicial padronizada conforme cursos ativos do workspace.

## Estrutura oficial (v1)
Objetivo: sair de uso casual para operação profissional com Claude Code em fluxo real de entrega.

Módulos:
1. Fundamentos e setup do Claude Code
2. Fluxo de execução guiada (prompt -> plano -> implementação)
3. Qualidade técnica (testes, lint, review, commit discipline)
4. Contexto e memória de projeto (arquivos de contrato e continuidade)
5. Operação no dia a dia (debug, refactor, documentação)
6. Automação e escala (tarefas repetíveis, playbooks, integração)
7. Capstone (entrega ponta a ponta com critérios objetivos)

## Princípios pedagógicos
- Cada módulo fecha em: problema real -> modelo mental -> execução -> validação.
- Foco em evidência prática (artefato verificável), não só teoria.
- Progressão por complexidade: fundamentos -> rotina -> escala.
- Qualidade mínima inegociável: testes + lint + documentação essencial.

## Próximos passos
1. Definir outcomes e critérios de aprovação por módulo.
2. Converter módulos em aulas graváveis (15–25 min).
3. Padronizar checklist/rubrica/DoD por aula.
4. Definir projeto transversal do curso (capstone com checkpoints).
