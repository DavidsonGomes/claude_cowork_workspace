# Financeiro

Controle financeiro pessoal e da EVO Foundation.

---

## Como funciona

- Use o plugin **finance** para análises, demonstrativos e controles.
- Documentos e planilhas ficam nesta pasta.
- Relatórios mensais gerados com o plugin **xlsx**.

## O que gerenciar aqui

- Fluxo de caixa mensal
- Análise de variações orçamentárias
- Demonstrativos financeiros (P&L, balanço)
- Controle de receitas e despesas
- Fechamento mensal

## Comandos úteis

| Diga isto | O que acontece |
|---|---|
| "analisa variação do orçamento" | Claude faz análise de variação |
| "cria demonstrativo financeiro" | Claude gera o demonstrativo |
| "prepara fechamento do mês" | Claude estrutura o processo de fechamento |
| "cria planilha financeira" | Claude gera planilha Excel |
| "reconcilia [conta]" | Claude ajuda na reconciliação |

## Rituais recomendados

- **Fechamento mensal**: revisar receitas, despesas e fluxo de caixa
- **Revisão trimestral**: comparar com orçamento e ajustar projeções

## Integração

- **finance** plugin → análises, demonstrativos, journal entries
- **xlsx** skill → planilhas financeiras
- **pdf** skill → relatórios em PDF
