# Redes Sociais

Estratégia e produção de conteúdo para redes sociais do Davidson e da EVO Foundation.

---

## Como funciona

- Use o plugin **marketing** para criar e planejar conteúdo.
- Use o plugin **brand-voice** para manter consistência de tom e voz.
- Rascunhos e calendário ficam nesta pasta.

## Canais ativos

*(Preencha com seus canais: LinkedIn, Twitter/X, Instagram, YouTube, etc.)*

## Cadência recomendada

- Definir temas da semana toda segunda-feira
- Produzir conteúdo em lotes (1-2x por semana)
- Revisão de métricas toda sexta

## Comandos úteis

| Diga isto | O que acontece |
|---|---|
| "cria post sobre [tema]" | Claude redige o post para o canal escolhido |
| "planeja conteúdo da semana" | Claude sugere pauta semanal |
| "cria calendário de conteúdo" | Claude monta o calendário editorial |
| "revisa esse post" | Claude revisa com base na brand voice |
| "cria sequência de emails sobre [tema]" | Claude monta sequência completa |

## Integração

- **Canva** (MCP conectado) → criação de artes e visuais
- **brand-voice** plugin → manter voz consistente
- **marketing** plugin → estratégia, campanhas, SEO

## Brand Voice

*(Use o comando "discover brand" ou "generate guidelines" para criar seu guia de voz assim que tiver materiais.)*
