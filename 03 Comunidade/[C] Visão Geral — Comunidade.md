# Comunidade

Gestão da comunidade open source e de usuários da EVO Foundation.

---

## Como funciona

- Documentação e planejamento ficam aqui e no **Notion** (MCP conectado).
- Use o plugin **product-management** para stakeholder updates e síntese de feedback.
- Use o plugin **marketing** para comunicações e campanhas para a comunidade.

## O que gerenciar aqui

- Calendário de eventos e meetups da comunidade
- Updates e comunicados para membros
- Síntese de feedback e solicitações da comunidade
- Planejamento de iniciativas open source

## Comandos úteis

| Diga isto | O que acontece |
|---|---|
| "cria update para a comunidade sobre [tema]" | Claude redige o comunicado |
| "sintetiza o feedback da comunidade" | Claude organiza os pontos principais |
| "planeja evento de comunidade" | Claude ajuda a estruturar o evento |

## Integração

- **Notion** → base de conhecimento da comunidade
- **Gmail** → comunicações com membros
- **Google Calendar** → eventos e meetups
