# Tarefas

Área central de gestão de tarefas e to-dos do Davidson.

---

## Como funciona

- As tarefas são gerenciadas via **Linear** (MCP conectado) e pelo arquivo `TASKS.md` local.
- Use o plugin **productivity** para criar, listar e atualizar tarefas.
- Faça uma revisão semanal toda segunda-feira.

## Comandos úteis

| Diga isto | O que acontece |
|---|---|
| "quais são minhas tarefas?" | Claude lista suas tarefas abertas |
| "adiciona tarefa: [descrição]" | Claude cria a tarefa |
| "o que está pendente?" | Claude filtra tarefas não concluídas |
| "revisão semanal" | Claude organiza e prioriza a semana |

## Integração

- **Linear** → issues e projetos de desenvolvimento
- **TASKS.md** → tarefas gerais e pessoais (arquivo nesta pasta)

## Regras

- Toda tarefa deve ter: descrição clara, área relacionada, prioridade (alta/média/baixa)
- Tarefas de desenvolvimento vão no Linear; tarefas gerais ficam aqui
