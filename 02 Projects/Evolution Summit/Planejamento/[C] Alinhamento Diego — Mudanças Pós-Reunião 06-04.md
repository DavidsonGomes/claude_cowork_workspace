## Diego, segue o que mudou após a reunião de hoje:

### 1. Pricing — Pivot para 3 Tiers

O modelo de 2 tiers (Contributor R$89 + Pro Support R$399) foi questionado. A decisão é migrar para **3 tiers: Free / Mensal / Anual**.

- **Free** serve para construir base engajada e monetizar depois
- **Anual** é o foco principal de venda
- R$399-499/mês foi apontado como barreira alta demais para o público open-source (freelancers, devs)
- Pricing precisa cobrir custos (suporte, tokens IA) e oferecer desconto atrativo no anual

**Precisa definir juntos:**
- Valores exatos de cada tier
- O que entra no Free (só Discord? acesso limitado à ferramenta?)
- Meta de receita revisada (antes: R$150k de ~300 users)
- Se os bônus de lançamento (Academy 6 meses, Blueprint, Fast Start) continuam válidos

### 2. Posicionamento da Marca

Saímos de "IA + automação multicanal" para **"Marketing Automation"**.

- Escopo mais amplo que permite expandir para outras ferramentas (ex: email marketing open-source) sob a marca Evo
- Isso muda a narrativa do Summit e dos conteúdos de lançamento

### 3. Comunidade — Modelo Híbrido

- **Free** → fica no Discord (engajamento diário alto)
- **Pago** → migra para plataforma Evo (controle de dados, SLA de suporte)

No documento atual não existe tier gratuito. Precisa ser incorporado.

### 4. Academy — Pipeline Estruturado

- Não depender só de lives que viram aulas (modelo Sobral)
- Criar **cursos estruturados por tema** (ex: "Cloud Code do Zero ao Avançado")
- Usar **multistream** (YouTube/Twitch/TikTok) para live coding, depois editar em módulos polidos

### 5. Metas e Métricas

Preciso revisar com você as metas de conversão e receita considerando o novo modelo de 3 tiers. O cenário muda bastante.
